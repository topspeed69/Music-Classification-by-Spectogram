# Mel Spectogram Output Directory

This directory contains generated Mel spectrogram images converted from the audio dataset.

## Contents
- **.png files**: Mel spectrogram representations of the audio tracks.
- **Note**: The actual image files are ignored by git to save space. Only this README is tracked.

## Usage
Run the `audio_to_spectogram_mel.py` script to populate this directory:
```bash
python AudioToSpectogram/audio_to_spectogram_mel.py
```
